Spectacle Lake, WA

Above is a simple map using the Google Maps API. It is depicting Spectacle Lake in Washington (which is the image I used as a background for my website). It is one of my favorite hikes along the Pacific Crest Trail. Google's Map API allows a variation of customizable options, you just have to manipulate the code you use to display the web map within your website. In my example I chose to display the terrain to help visualize the landscape. There are several other basic map types that Google Maps API has built in.
